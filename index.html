Basics

This is an editor for two-dimensional programs made of glyphs, each of which has an alias (plain-text name) and an effect. You can change a glyph by clicking on it and entering a new alias, or clear it by leaving the input empty. The current program is stored in the hash fragment of the URL, allowing for easy sharing and undo/redo.

Program execution is controlled by a cursor, which starts at the top left cell and moves to the right, executing each glyph it passes over. Some glyphs change the direction of the cursor, facilitating control flow, while others modify the stack or perform I/O.
